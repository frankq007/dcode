### ArkTS 鸿蒙原生 Markdown 渲染解决方案

### 版本更新描述

v3.4.3版本

1. 支持 Mermaid 图表渲染
2. 新增 `setMermaidEnable()` 接口，设置是否开启 mermaid 渲染
3. 新增 `setMermaidTheme()` 接口，设置 Mermaid 深浅色主题
4. 新增 `setMermaidSecurityLevel()` 接口，设置 Mermaid 预览安全级别
5. 新增 `setQuoteTextColor()` 接口，设置引用块文本颜色
6. 修复渲染出来的任务列表可交互的问题

---

v3.4.2版本

1. 新增 `setLazyRender()` 接口，支持设置是否开启可见区域懒渲染，优化大文档加载性能
2. 新增 `setLazyPreloadBlockCount()` 接口，设置懒渲染预渲染块数量
3. 新增 `setSetextHeadingEnable()` 接口，支持设置是否开启 Setext 标题解析
4. 新增 `setTableTitleTextColor()` 接口，支持设置表格表头文本颜色
5. 新增 `setTableInterleaveTextColor()` 接口，支持设置表格错行文本颜色
6. 新增 `setTableNormalTextColor()` 接口，设置表格非错行文本颜色
7. 修复 $ 符号解析金额时异常的问题

---

v3.4.1版本

1. 支持 Setext 标题语法
2. 修复部分复杂超链接无法被正确解析的问题
3. 修复在禁用文本复制时抢占外部 .gesture(LongPressGesture()) 的问题

---

v3.4.0版本

1. 新增实例级 `plugins` 配置扩展自定义语法，并将匹配结果渲染为对应组件。
2. 新增 `setLatexClickListener()` 接口，支持在公式图片渲染完成后获取公式原文与 `pixelMap`
3. 新增对 `<font color size face>` 行内快速样式的解析支持
4. 新增跨 Text 文本选择能力，支持长按拖拽选择并通过监听回调交由业务侧执行复制
5. 新增 `setTextSelectionEnable()` 接口，设置是否开启组件的长按选择能力，默认关闭；开启后，可通过长按 Markdown
   文本并拖拽两侧光标的方式选择跨段落内容。
6. 新增 `setTextSelectionCopyListener()` 接口，设置文本选择后的复制监听。因系统剪贴板涉及业务权限控制，组件只回传最终选中的文本内容，由业务侧决定是否写入剪贴板。
7. 修复模块名非 entry 时，渲染 LaTeX 公式时，公式显示为乱码的问题
8. 弃用 `setCopyOption()` 接口
